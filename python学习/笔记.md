# 三，数组，增删改查

```python
1,添加
    insert(索引位置,值)
    append(值) 数组追加单个值
    extend([值1,值2]) 数据追加数组
    实例
        member = ['小甲鱼', '汉云', '伴你']
        print(member)
        member.append('python学习')
        print(member)

        member.extend(['python数据', '网络获取数据'])
	    print(member)

        member.insert(0,'追加第一个位置')
        print(member)
 2，删除
	del arr[索引]/arr  删除指定的元素或者删除整个数组
    remove('元素值')
    pop()剔除最右边，pop(索引)根据索引踢出
 3，数组分片
	member[1:3] 取到数组里面从索引为1到3的数据，包括1不包括3，并且不改变原来数组
 4，排序
	sortL = [1, 3.4, 7, 8, 8, 19]
    sortL.sort()
    print(sortL)
    倒叙
    	sortL.sort(reverse=True)
 5, 反转
	sortL = [1, 3.4, 7, 8, 8, 19]
    sortL.reverse()
    print(sortL)
	
		
```

## 1，字符串基础方法

```python
str1 = "i like study python"
print(str1)
# 首字母大写
str2 = str1.capitalize()
print(str2)
# 所有字母变小写
str3 = "LLDFLFKJD"
print(str3.casefold())
# 左右俩测添加空格
str4 = str3.center(40)
print('7' + str4 + '7')
# 检查子串在父串中的次数
str2 = "skfksdlslldfas"
print(str2.count("l"))
str2 = 'skfksdlslldfas'
print(str2.join("12345"))
print(str2)
```
