条件判断和逻辑判断

```python
1，条件判断
	if(条件):
        print("if")
    else:
        print("else")
    或者
    if(条件):
        print("if")
    elif(条件):
        print("elif")
    else:
        print("else")
2,循环语句
	while 条件 and 条件:
        print("while")
3，预算符号
	**：幂次运算 2**3 （2到3次方） //：除号得到小数点以后会舍弃
    + - * /  加减乘除
4，逻辑运算
	not and or
5，for循环
	temp = "iloveyou"
    for i in temp:
        print(i, end="+")

    arr = ["Irving", "love", "wumengxiang"]
    for each in arr:
        print(each, len(each))
        
 6，range()跟for循环一起使用
	list_value = list(range(9))
	for i in list_value:
    	print(i, end="<>")
     每次循环步数为3
    	for i in range(1,9,3):
    		print(i)
```

数组，增删改查

```python
1,添加
    insert(索引位置,值)
    append(值) 数组追加单个值
    extend([值1,值2]) 数据追加数组
    实例
        member = ['小甲鱼', '汉云', '伴你']
        print(member)
        member.append('python学习')
        print(member)

        member.extend(['python数据', '网络获取数据'])
	    print(member)

        member.insert(0,'追加第一个位置')
        print(member)
 2，删除
	del arr[索引]/arr  删除指定的元素或者删除整个数组
    remove('元素值')
    pop()剔除最右边，pop(索引)根据索引踢出
 3，数组分片
	member[:]
		
```
